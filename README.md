# Solidity Starter

This is a template repo for starting solidity projects.

## Installation

Clone the repo

```bash
git clone https://github.com/cunningham32/solidity-starter.git
```

Install dependencies
```bash
npm install
```

## Usage

Compile contracts
```bash
npx hardhat compile
```

Test the contracts
```bash
npx hardhat test
```
